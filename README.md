# select-sample — QUINZE (패션 셀렉트샵)

이피웹(YIPI WEB) 포트폴리오용 ⑤-1번 샘플. 패션 셀렉트샵.

- **매트릭스 셀:** 분할/게이트 진입 · 제품그리드/포폴 주인공 · **숫자 인덱스 내비** · 호버 리빌 · **하이콘트라스트 볼드/브루탈리즘**
- **차별점:** 첫 게이트 진입(입장 오버레이) + 첫 숫자 인덱스 내비 + 첫 호버 리빌 + 첫 브루탈리즘 무드(페이퍼+잉크+시그널 레드). 마퀴 티커.
- **타입:** Anton(헤비 디스플레이) + Pretendard.
- **정직 라벨:** 가짜 결제·장바구니 없음 → 제품은 외부 스토어(네이버 스마트스토어)·인스타 연결. 홀세일/입점은 "문의" 폼(Formspree). 모바일 하단바 = 인스타·스토어·문의(PC CTA와 통일).
- **앵커 보정:** `scroll-padding-top` 적용.

배포: Cloudflare Workers. Formspree `YOUR_FORM_ID` 치환 필요.
